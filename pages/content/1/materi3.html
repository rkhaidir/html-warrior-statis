Tag Syntax dan Option

Pada dasarnya HTML berisi kode-kode tertentu berbasis teks yang disebut "tag". Tag bisa diartikan sebagai tanda khusus yang akan dibaca oleh browser dan menampilkannya kepada pengguna. Berikut jenis-jenis syntax dan option HTML yang sering digunakan:

Tag Komentar

Untuk membuat atau menambahkan komentar kita hanya perlu mengikuti sintaks berikut ini:
<!-- Komentar disini -->

Perhatikan pada tag pembuka komentar terdapat “tanda seru (!)”, tapi tidak pada tag penutup. Komentar tidak akan ditampilkan oleh browser, komentar hanya digunakan untuk debuging maupun membantu programmer untuk memberi keterangan dari suatu source code.

Tag Images

Tag image didefinisikan dengan tag <img>. Untuk memasukan gambar gunakan atribut src, alternatif teks gunakan atribut alt, mengatur lebar gambar gunakan atribut width, dan untuk mengatur tinggi gambar gunakan atribut height.
