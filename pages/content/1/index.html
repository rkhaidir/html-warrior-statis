Struktur Dasar HTML

HTML adalah singkatan dari Hypertext Markup Language. HTML memungkinkan seorang pengguna untuk membuat dan menyusun bagian paragraf, heading, link atau tautan, tabel dan elemen lainnya untuk membuat halaman web dan aplikasi.

HTML bukanlah bahasa pemrograman, dengan begitu HTML tidak punya kemampuan untuk membuat fungsionalitas yang dinamis. Sebagai gantinya, HTML memungkinkan pengguna untuk mengorganisir dan memformat dokumen, sama seperti Microsoft Word.

Setiap halaman HTML minimal memiliki struktur dasar yang terdiri atas DTD atau DOCTYPE, tag html, tag head dan tag body:

  • DTD atau DOCTYPE
    Tag paling awal dari HTML adalah DTD atau DOCTYPE (<!DOCTYPE html>). DTD adalah singkatan dari Document Type Declaration yang berfungsi untuk memberitahu web browser bahwa dokumen yang akan diproses adalah dokumen HTML5.

  • Tag <html>
    Tag <html> merupakan tag pembuka dari keseluruhan halaman web. Semua kode html harus berada di dalam tag ini.

  • Tag <head>
    Elemen pada tag <head> umumnya berisi berbagai definisi halaman seperti kode CSS, Javascript, tag meta, viewport dan kode-kode lainnya yang tidak tampil di browser.

  • Tag <body>
    Tag ini berisi semua elemen yang akan tampil di halaman web seperti paragraf, tabel, link, gambar, list dan masih banyak lagi. Sebagian besar waktu dalam merancang atau mendesain web akan dihabiskan di dalam tag <body> ini.
