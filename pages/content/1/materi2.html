Versi HTML

Seiring berjalannya waktu dalam perkembangan dunia global khususnya dalam bidang teknologi berkembang sangat pesat, begitupun dengan HTML. HTML ikut berkembang dan berpartisipasi untuk memajukan dunia IT dalam bidang Web App. Hal yang perlu dipahami adalah setiap ada perkembangan dan perubahan versi dari HTML, mau tidak mau browser pun harus mengupgrade diri agar bisa mendunkung dari pembaharuan HTML yang dilakukan. Berikut versi HTML dari awal hingga sekarang:

  • Versi HTML1 (1991)
    Versi HTML1 merupakan versi awal yang hanya bisa digunakan untuk membuat heading, hypertext, cetak tebal, cetak miring, list dan paragraf. Pada versi ini, sebuah web hanya berisi satu atau dua gambar saja.

  • Versi HTML2.0 (1995)
    Pada versi ini dapat dikatakan sebagai awal untuk membuat sebuah web interaktif. Terdapat penambahan beberapa fitur, salah satunya adalah fitur form.

  • Versi HTML3.2 (1997)
    Banyak penambahan fitur yang lebih fleksibel untuk membuat web interaktif. Salah satu fitur yang paling terkenal pada versi ini adalah fitur tabel di dalam paragraf. Versi HTML3.2 adalah penyempurnaan dari Versi HTML3.

  • Versi HTML4.01 (1999)
    Pada versi ini web sudah semakin interaktif dan lebih mementingkan UI dan UX. Pembahruan ini hampir terjadi di segala perintah-perintah HTML seperti table, img, link, meta, form dan lainnya. Secara umum, HTML4.01 merupakan perbaikan dari Versi HTML4.

  • XHTML (2000)
    XHTML merupakan singkatan dari eXtensible HyperText Markup Language. XHTML merupakan bentuk reformulasi dari HTML menggunakan paradigma XML.

  • Versi HTML5 (2014)
    HTML5 merupakan versi HTML yang diperbaharui hingga sekarang dan menjadi standar terbaru sebagai proyek kerja antara W3C (World Web Web Consurtium) dan WHATG (Web Hypertext Application Technology Group). Berikut 5 fitur menarik dari HTML5:

    1. Video dan elemen audio untuk pemutaran.
    2. Unsur Canvas untuk menggambar.
    3. Elemen baru seperti footer, header, dan nav.
    4. Dukungan lebih baik untuk penyimpanan (Local Storage).
    5. Kontrol bentuk baru seperti kalender, tanggal, waktu, url dan search.
Berikut perkembangan atau evolusi dari web, tidak hanya HTML. Semua aspek, mulai dari http, web browser, internet, javascript, css, kuki dan masih banyak lagi.
