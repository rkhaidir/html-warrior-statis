Latihan Format Teks pada Halaman Web

Soal:

Lengkapi isian kosong, dengan dokumen HTML sederhana!


<html>
<head>
Page Title</title>
</head>

</body>

Latihan Format Teks pada Halaman Web

Soal:

Lengkapi isian kosong, dengan Tag komentar dan Tag heading paling penting!

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
Isi dengan tag komentar
Isi dengan heading paling penting/besar.

</body>
</html>
